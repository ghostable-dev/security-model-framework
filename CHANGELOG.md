# Changelog

All notable changes to this project will be documented in this file.

This project follows a simple versioning approach while the convention is early:

- Patch releases: wording fixes, small checklist improvements, examples.
- Minor releases: new templates, new audit sections, meaningful structure changes.
- Major releases: breaking convention or filename changes.

## [0.1.0] - 2026-04-24

### Added

- Initial `SECURITY_MODEL.md` convention.
- Generic web application template.
- Laravel-specific security model template.
- GitHub-compatible `SECURITY.md` vulnerability policy template.
- AI agent security audit guide.
- Laravel security model author skill.
- Laravel security review skill with hotspot scanner.
- Laravel security test writer skill.
- Package-specific Laravel security references for Livewire, Sanctum, Passport, Scout, Cashier, Filament, Nova, Horizon, Telescope, and Spatie Permission.
- Generic web app example.
- Laravel SaaS example.
- Open-source support files.
